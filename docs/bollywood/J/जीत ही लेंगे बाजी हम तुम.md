# जीत ही लेंगे बाज़ी हम तुम

दीपक को परवाना</br>
दुनिया अपने रब को पुकारे</br>
तुझको तेरा दीवाना</br>

आओ वही हम जाये</br>

इस दुनिया को ठुकराएँ</br>

जिसको ज़माना लूटे ना</br>

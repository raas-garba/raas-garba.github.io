# છાનું રે છપનું

છાનું રે છપનું કંઇ થાય નઇ</br>
ઝમકે ના ઝાંઝર તો ઝાંઝર કહેવાય</br>
એક ઘાયલ ને પાયલ બે છૂપ્યા છૂપાય નઇ</br>
ઝાંઝરને સંતાડી રાખ્યું રખાય નઇ

આંખ્યો બચાવી ને આંખના રતનને</br>
પરદામાં રાખીને સાસુ નણંદને</br>
ચંપાતા ચરણો એ મળ્યુ મળાય નઇ

નણદી ને નેપૂર બે એવા અનાડી</br>
વ્હાલા પણ વેરી થઇ ખાય મારી ચાડી</br>
આવેલા સપનાનો લ્હાવો લેવાય નઇ

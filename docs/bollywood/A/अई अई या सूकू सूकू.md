# अई अई या सूकू सूकू

अई अई या सूकू सूकू</br>
अई अई या करूँ मैं क्या, सूकू सूकू</br>
खो गया दिल मेरा, सूकू सूकू</br>
कोई मस्ताना कहे, कोई दीवाना कहे</br>
कोई परवाना कहे, जलने को बेक़रार</br>
चाहत का मुझपे असर, मैं दुनिया से बेख़बर</br>
चलता हूँ अपनी डगर, मंज़िल है मेरी प्यार</br>
अई अई या सूकू सूकू</br>
अई अई या करूँ मैं क्या ...

चौंको मत ऐ दोस्तो, यूँही मुझसे मत डरो</br>
मेरे भोले प्यार को नादानी ना समझो</br>
हाथों में जो हाथ है, जीवनभर का साथ है</br>
सीधी-सच्ची बात है, कहानी ना समझो</br>
अई अई या सूकू सूकू</br>
अई अई या करूँ मैं क्या ...

मेरी आँखों में नशा, मेरी बातों में नशा</br>
मेरी साँसों में नशा, बहका मैं बिन पिये</br>
घुटती-घुटती बेबसी, सहमी-सहमी हर घड़ी</br>
बिन चाहत क्या ज़िंदगी, जीता मैं किसलिए</br>
अई अई या सूकू सूकू</br>
अई अई या करूँ मैं क्या

# तुम पास आये

तुम पास आये, यूँ मुस्कुराये</br>
तुमने न जाने क्या सपने दिखाये</br>
अब तो मेरा दिल, जागे न सोता है</br>
क्या करूँ हाय, कुछ कुछ होता है ...

न जाने कैसा एहसास है</br>
बुझती नहीं है क्या प्यास है</br>
क्या नशा इस प्यार का</br>
मुझपे सनम छाने लगा</br>
कोई न जाने क्यों चैन खोता है</br>
क्या करूँ हाय, कुछ कुछ होता है ...

क्या रंग लायी मेरी दुआ</br>
ये इश्क़ जाने कैसे हुआ</br>
बेचैनियों में चैन</br>
न जाने क्यों आने लगा</br>
तनहाई में दिल, यादें संजोता है</br>
क्या करूँ हाय, कुछ कुछ होता है ...

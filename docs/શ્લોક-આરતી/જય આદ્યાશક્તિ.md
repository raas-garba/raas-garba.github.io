# જય આદ્યાશક્તિ

જય આદ્યાશક્તિ, મા જય આદ્યાશક્તિ</br>
અખંડ બ્રહ્માંડ નિપજાવ્યાં, પડવે પ્રગટ્યાં,</br>
મા જયો જયો મા જગદંબે ...

દ્વિતીયા બેઉ સ્વરૂપ શિવ શક્તિ જાણું,</br>
બ્રહ્મા ગણપતિ ગાયે, હર ગાયે દર મા

તૃતીયા ત્રણ સ્વરૂપ, ત્રિભુવનમાં બેઠાં,</br>
ત્રયા થકી તરવેણી, તું તરવેણી મા

ચોથે ચતુરા મહાલક્ષ્મી, મા સચરાચર વ્યાપ્યાં,</br>
ચાર ભુજા ચો દિશા, પગટ્યાં દક્ષિણમાં

પંચમી પંચ ઋષિ, પંચમી ગુણ પદમા,</br>
પંચ તત્ત્વ ત્યાં સોહીએ, પંચે તત્ત્વો મા

ષષ્ઠી તું નારાયણી મહિષાસુર માર્યો,</br>
નરનારીના રૂપે, વ્યાપ્યાં સઘળે મા

સપ્તમી સપ્ત પાતાળ, સંઘ્યા સાવિત્રી,</br>
ગૌ ગંગા ગાયત્રી, ગૌરી ગીતા મા

અષ્ટમી અષ્ટ ભુજા, આઈ આનંદા,</br>
સુરનર મુનિવર જનમ્યા, દેવો દૈત્યો મા

નવમી નવકુળ નાગ, સેવે નવ દુર્ગા,</br>
નવરાત્રીનાં પૂજન, શિવરાત્રીનાં અરચન,</br>
કીધાં હર બ્રહ્મા

દશમી દશ અવતાર, જય વિજયા દશમી,</br>
રામે રામ રમાડ્યાં, રાવણ રોળ્યો મા

એકાદશી અગિયારસ, કાત્યાની કામા,</br>
કામ દુર્ગા કાલિકા, શ્યામા ને રામા

બારશે બાળા રૂપ બહુચરી અંબા મા,</br>
બટુક ભૈરવ સોહે, કાળ ભૈરવ સોહે,</br>
તારા ચાચરમાં

તેરસે તુળજા રૂપ તું તારુણી માતા,</br>
બ્રહ્મા વિષ્ણુ સદાશિવ, ગુણ તારા ગાતા

ચૌદશે ચૌદા રૂપ, ચંડી ચામુંડા,</br>
ભાવ ભક્તિ કાંઈ આપો, ચતુરાઈ કાંઈ આપો</br>
સિંહવાહિની મા

પૂનમે કુંભ ભર્યો સાંભળજો કરુણા,</br>
વસિષ્ઠદેવે વખાણ્યાં, માર્કંડમુનિએ વખાણ્યાં,</br>
ગાઈ શુભ કવિતા

સંવત સોળ સત્તવન, સોળસે બાવીસ મા,</br>
સંવત સોળે પ્રગટ્યાં, રેવાને તીરે,</br>
ગંગાને તીરે

ત્રંબાવટી નગરી આઈ રૂપાવટી નગરી,</br>
સોળ સહસ્ત્ર ત્યાં સોહીએ, ક્ષમા કરો ગૌરી,</br>
દયા કરો ગૌરી

શિવ શક્તિની આરતી જે કોઈ ગાશે,</br>
મા જે ભાવે ગાશે,</br>
ભણે શિવાનંદ સ્વામી,</br>
સુખ-સંપત્તિ થાશે,</br>
હર કૈલાશે જાશે,</br>
મા અંબા દુખ હરશે,</br>
ૐ જયો જયો મા જગદંબે ...

જય આદ્યાશક્તિ, મા જય આદ્યાશક્તિ</br>

મા જયો જયો મા જગદંબે ...

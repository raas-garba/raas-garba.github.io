# મારા ઘટમાં બિરાજતા

મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી મહાપ્રભુજી</br>
મારુ મનડું છે ગોકુલ વનરાવન</br>
મારા તનના આંગિણયાંમાં તુલસીના વન</br>
હે મારા પ્રાણ જીવન...

મારા આતમના આંગણે શ્રીમહાપ્રભુજી</br>
મારી આંખો વશે ગિરધારી રે ધણી</br>
મારુ તન મન ગયું છે જેને વારી રે વારી</br>
હે મારા શ્યામ મોરારિ...</br>
મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી શ્રીમહાપ્રભુજી...

હું તો નિત્ય વિઠ્ઠલ વર ની સેવા રે કરું</br>
હું તો આઠે સમા કેરી ઝાંખી રે કરું</br>
મેં તો ચિતડું શ્રીનાથજી ને ચરણે ધર્યું</br>
જીવન સફળ કર્યું...</br>
મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી શ્રીમહાપ્રભુજી...

હે મારા પ્રાણ થકી મને વૈષ્ણવ વ્હાલા</br>
નિત્ય કરતા શ્રીનાથજી ને કાલા રે વાલા</br>
મે તો વલ્લભ પ્રભુજી ના કીધાં છે દર્શન</br>
મારું મોહી લીધું મન...</br>
મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી શ્રીમહાપ્રભુજી...

મેં તો ભક્તિ રે મારગ કેરો સંગ રે સાધ્યો</br>
મેં તો પુષ્ટિ રે મારગ કેરો સંગ રે સાધ્યો</br>
મને ધોળ કિર્તન કેરો રંગ રે લાગ્યો</br>
મેં તો લાલાની લાલી કેરો નંગ રે માંગ્યો</br>
હીરલો હાથ લાગ્યો...</br>
મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી શ્રીમહાપ્રભુજી...

મારી અંત સમય કેરી સુણો રે અરજી</br>
લે જો શ્રીજીબાવા શરણોમાં દયા રે કરી</br>
મને તેડાં રે યમ કેરાં કદી ન આવે</br>
મારો નાથ તેડાવે...</br>
મારા ઘટમાં બિરાજતા શ્રીનાથજી યમુનાજી શ્રીમહાપ્રભુજી...

# તુમસે લગી તાલી

તુમસે લગી તાલી મૈયા તુમસે લગી તાલી</br>
અંબા મા ને નોતરૂ દેવા હું તો ચાલી</br>
ચણીયો પહેર્યો, ચોળી પહેરી ભૂલ ગઇ સાડી</br>
અંબા મા એ મુજને બાવરી બનાવી ...તુમસે લગી

બહુચર મા ને નોતરૂ દેવા હું તો ચાલી</br>
ફુલ લીધાં હાર લીધા ભૂલ ગઇ વેણી</br>
બહુચર મા એ મુજને બાવરી બનાવી ...તુમસે લગી

કાળકા મા ને નોતરૂ દેવા હું તો ચાલી</br>
ખીચડીમા ખાંડ નાખી, ખીર કરી ખારી</br>
કાળકા મા એ મુજને બાવરી બનાવી ...તુમસે લગી

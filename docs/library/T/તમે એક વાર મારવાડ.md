# તમે એક વાર મારવાડ

તમે એક વાર મારવાડ જાજો રે ઓ મારવાડા</br>
તમે મારવાડની મહેંદી લેતા આવજો રે ઓ મારવાડા

તમે ઓલ્લું લાવજો, પેલ્લું લાવજો</br>
પાન સોપારી પાનના બીડાં</br>
કચકડાની ડાબલી રે

તમે એક વાર સુરત જાજો રે ઓ મારવાડા</br>
તમે સુરતની ઘારી લેતા આવજો રે ઓ મારવાડા

તમે ઓલ્લું લાવજો, પેલ્લું લાવજો</br>
પાન સોપારી પાનના બીડાં</br>
કચકડાની ડાબલી રે

તમે એક વાર પાટણ જાજો રે ઓ મારવાડા</br>
તમે પાટણથી પટોળા લેતા આવજો રે ઓ મારવાડા

તમે ઓલ્લું લાવજો, પેલ્લું લાવજો</br>
પાન સોપારી પાનના બીડાં</br>
કચકડાની ડાબલી રે

તમે એક વાર જામનગર જાજો રે ઓ મારવાડા</br>
તમે જામનગરનું લેરિયું લેતા આવજો રે ઓ મારવાડા

તમે ઓલ્લું લાવજો, પેલ્લું લાવજો</br>
પાન સોપારી પાનના બીડાં</br>

# તું કાળી ને કલ્યાણી રે મા

તું કાળી ને કલ્યાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું ચારે યુગમાં ગવાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા

તને પહેલાં તે યુગમાં જાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું શંકરની પટરાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું ભસ્માસુર હરનારી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા

તને બીજા તે યુગમાં જાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું હરિશ્ચંદ્ર ઘરે પટરાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા

તને ત્રીજા તે યુગમાં જાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું રામચંદ્ર ઘેર પટરાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું રાવણને રોળનારી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા

તને ચોથા તે યુગમાં જાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું પાંડવ ઘેર પટરાણી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા</br>
તું કૌરવકુળ હણનારી રે મા, જ્યાં જોઉં ત્યાં જોગમાયા

# તારા વિના વેરણ

તારા વિના વેરણ લાગે આ રાતડી</br>
વાટ જુવે છે મારી આંખડી ... (૨)</br>
હે કાના... હું તને ચાહું (૨)

કેમ રે નિભાવશું પ્રીત રે ગોપાલા</br>
તું તો જગત નો સ્વામી બ્રિજની હું બાલા</br>
હે કાના... હું તને ચાહું (૨)

જનમોજનમની વ્હાલા હું તારી દામી</br>

હે કાના... હું તને ચાહું (૨)

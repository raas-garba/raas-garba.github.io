# ઝુલે ઝુલે ગબ્બરની માય

ઝુલે ઝુલે ગબ્બરની માય, અંબા ઝુલે છે</br>
માને ઝુલે ઝુલવાની હોંશ ઘણી</br>
ભક્તો ઝુલાવે ખમ્મા, મા ખમ્મા ઘણી</br>
ભક્તો ઝુલાવે, મા ખુશ થાય, અંબા ઝુલે છે</br>
ઝુલે ઝુલે ગબ્બરની માય, અંબા ઝુલે છે

માને દરવાજે નોબત ગડગડે છે</br>
વળી શરણાઇનાં સૂર સાથે મળે</br>
રસ મસ્તાનમાં સૂર સંભળાય, અંબા ઝુલે છે

મા તો ચાલે ત્યાં કુમકુમનાં પગલા પડે</br>
અંબા બોલે ત્યાં મુખડેથી ફુલડાં ઝરે</br>
વરસે વરસે કુમકુમનો વરસાદ, અંબા ઝુલે છે

આજ શોભા Charlotteની નવલી બની</br>
આવો ગાવો સૌ ભક્તજનો સાથે મળી</br>
ગરબો ગાયો, ગવરાવે મસ્તાન, અંબા ઝુલે છે

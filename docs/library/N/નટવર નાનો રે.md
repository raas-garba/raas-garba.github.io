# નટવર નાનો રે

નટવર નાનો રે કાનો રમે છે મારી કેડમાં

નંદકુંવર શ્યામકુંવર લાલકુંવર</br>
ફુલકુંવર નાનો રે ગેડીદડો કાનાના હાથમાં</br>
નટવર નાનો રે.....

ક્યો તો ગોરી ચિત્તળની ચૂંદડી મંગાવી દઉં</br>
ચૂંદડીનો વહોરનાર રે કાનો રમે છે મારી કેડમાં</br>
નટવર નાનો રે.....

ક્યો તો ગોરી નગરની નથડી મંગાવી દઉં</br>
નથડીનો વહોરનાર રે કાનો રમે છે મારી કેડમાં</br>
નટવર નાનો રે.....

ક્યો તો ગોરી ઘોઘાના ઘોડલા મંગાવી દઉં</br>
ઘોડલાનો વહોરનાર રે કાનો રમે છે મારી કેડમાં</br>
નટવર નાનો રે.....

ક્યો તો ગોરી હાલારના હાથીડા મંગાવી દઉં</br>
હાથીડાનો વહોરનાર રે કાનો રમે છે મારી કેડમાં</br>
નટવર નાનો રે.....

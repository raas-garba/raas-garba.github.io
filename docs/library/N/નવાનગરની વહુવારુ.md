# નવાનગરની વહુવારુ

નવાનગરની વહુવારુ</br>
તારો ઘૂમટો મેલ</br>
વળ વાયુની વચમાં જોને</br>
મિસરી નમણી નાગરવેલ હો</br>
હે હેજી તારો ઘૂમટો મેલ

વાયરે ચઢીને ફુલ રૂમ ઝૂમતા</br>
વનવગડે વેરાતા ફુલડાં</br>
ફુલડે રમતી ફોરમડી</br>
તારું પડ્યું મેલ</br>
સપના વેરી રમતી તારી</br>
નીંદર રાણી આવી રે હો...</br>
હેજી તારો ઘૂમટો મેલ

ફાગણ આયો મારા આંગણે, રંગ લાયો લાયો</br>
ભીંજે ઓઢણી, ને ભીંજે રે કંચબો</br>
ભીંજે રે લહેરીયું ગુલાલ

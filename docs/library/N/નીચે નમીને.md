# નીચે નમીને

નીચે નમીને નમસ્કાર કરૂં છુ મારા ડુંગરવાળા દેવીને</br>
પાયે પડીને પ્રણામ કરૂ છું મારા ડુંગરવાળા દેવીને

ત્રાંબા લોટા જળ રે ભર્યા છે</br>
નાવણ કરતા જાઓને ...નીચે નમીને

ખીર પૂરી ને લાપસી રંધાવુ</br>
જમણ કરતા જાઓને ...નીચે નમીને

પાન સોપારીના બીડવા બંધાવુ</br>
મુખવાસ કરતા જાઓને ...નીચે નમીને

# લાલ રે ગુલાબનાં ફુલોની ચુંદડી

લાલ રે ગુલાબનાં ફુલોની ચુંદડી (૨)</br>
રંગે લાલમ લાલ રે</br>
લઈને તાડી ઘુમે ભવાની (૨), ગરબામાં ઉડે ગુલાલ રે

નોરતાની રાત રૂડી રાત રઢીયાળી (૩)</br>
ભક્તોની ભીડ ભાંગે દુર્ગા ભવાની (૨)

ચાચર ચોકમાં માનાં મંદિરીયે (૨)</br>
કિરણોની ચલતી મસાલ રે</br>
લઈને તાડી ઘુમે ભવાની (૨), ગરબામાં ઉડે ગુલાલ રે</br>
લાલ રે ગુલાબનાં ફુલોની ચુંદડી

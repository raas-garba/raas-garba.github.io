# બજે તાલ મંજીરા ઢોલ રે ભવાની મા

બજે તાલ મંજીરા ઢોલ રે ભવાની મા</br>
જોગણીયું સૌ ડોલે મનમાની મા</br>
ગબ્બરને હીંડોળે</br>
રમવા આવો ને મા, મારી જોડે

તોરણ બાંધ્યા શેરીને પોળે રૂપાળી મા</br>
મસ્તક તારે ખોળે બિરદાળી મા</br>
જનમ જનમને કોલે</br>
રમવા આવો ને મા, મારી જોડે

બજે તાલ મંજીરા ઢોલ રે ભવાની મા</br>
પહેરી પગમાં પાવડી</br>
તમે આવો ને રમવા માવડી</br>
છે અંતર આશ આવડી</br>
તમે તારો અમારી નાવડી

તુજ ભક્તિ ભરી રસ છોળે બિરદાળી મા</br>
તનમનિયા તરબોળે મતવાલી મા</br>
હૈયું ઝંખી ઝોળે</br>
રમવા આવો ને મા, મારી જોડે

બજે તાલ મંજીરા ઢોલ રે ભવાની મા

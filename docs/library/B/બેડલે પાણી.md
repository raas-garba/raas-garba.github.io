# બેડલે પાણી

બેડલે પાણી હો બેડલે પાણી</br>
મારા મનડાની વાત ક્યાં છુપાણી

મને પૂછો તો કાંઇના જાણું</br>
જુઓ રોમ રોમ એને હું માણું</br>
મારા હૈયાની કોર છે ભીંજાણી

પેલો ચાંદલિયો ઓઢણીએ ઝીલું</br>
કો'ક અણસારે ઓચિંતી ખીલું</br>
ઓલી અમરતની ધાર છે સિંચાણી

આજ સૈયર સૌ સાથ ઘૂમ ઘૂમું</br>
ભર્યા સપનાંએ રુમઝુમ ઝુમું</br>
નવી ફૂટે છે આજ નવી વાણી

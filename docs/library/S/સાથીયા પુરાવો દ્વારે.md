# સાથીયા પુરાવો દ્વારે

સાથીયા પુરાવો દ્વારે, દિવાડા પ્રગટાવો રાજ...</br>
આજ મારે આંગણે પધારશે મા પાવાવાળી..

વાંઝિયાનો મેણો ટાળી રમવા રાજકુમાર દે મા,</br>
ખોળાનો ખુંદનાર દે</br>
કુંવારી કન્યાને માડી મનગમતો ભરથાર દે મા,</br>
પ્રિતમજીનો પ્યાર દે

નિર્ધનને ધનધાન આપે, રાખે માડી સૌની લાજ</br>
આજ મારે આંગણે પધારશે મા પાવાવાળી..

કુમકુમ પગલા ભરશે માડી સાતે પેઢી તરશે</br>
આદ્યશક્તિ મા પાવાવાળી પીડા જનમ જનમની હરશે

દૈ દૈ તાળી ગાઓ આજ વાજિંત્રો વગડાવો રાજ</br>
આજ મારે આંગણે પધારશે મા પાવાવાળી..

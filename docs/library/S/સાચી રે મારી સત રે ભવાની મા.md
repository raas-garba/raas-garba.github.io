# સાચી રે મારી સત રે ભવાની મા

સાચી રે મારી સત રે ભવાની મા, અંબા ભવાની મા

હું તો તારી સેવા કરીશ મૈયા લાલ</br>
નવ નવ રાતના નોરતાં કરીશ મા, પૂજાઓ કરીશ મા</br>
વિરાટનો ગરબો તારો ઝીલીશ મૈયા લાલ</br>
સાચી રે મારી સત રે ભવાની મા, અંબા ભવાની મા

અંબાની ડાળે મા પારણાં બંધાવો મા</br>
હાલરડાં ગવરાવો મા</br>
પારણિયે પોઢે નાના બાળ મૈયા લાલ</br>
સાચી રે મારી સત રે ભવાની મા, અંબા ભવાની મા

આજે રે આજે મા લવીંગ સોપારી માં પાન સોપારી માં</br>
આજે રે કેવડીયાળા પાન મૈયા લાલ</br>
સાચી રે મારી સત રે ભવાની મા, અંબા ભવાની મા

# સુના સરવરીયા

સુના સરવરીયા ને કાંઠડે હું</br>
બેડલું મેલીને ન્હાવા ગઇ</br>
પાછી વળી ત્યારે બેડલુ નહી, બેડલુ નહી.

હું તો મન માં ને મન માં મુંઝાણી મારી સૈ,</br>
શું રે કહેવું રે મારે માવડી ને જૈ</br>
પાછી વળી ત્યારે બેડલુ નહી, બેડલુ નહી.....સુના સરવરીયા.....

કેટલુંયે કહ્યું પણ કાળજુ ના કોર્યું</br>
ને ચોરી ચોરી ને એણે બેડલું ચોર્યું</br>
ખાલીખમ બેડલાથી વળે ના કંઇ</br>
પાછી વળી ત્યારે બેડલુ નહી, બેડલુ નહી.....સુના સરવરીયા.....

નીતરતી ઓઢણી ને નીતરતી ચોળી</br>
ને બેડલા નો ચોર મારે કેમ લેવો ખોળી</br>
દઇ દે મારું બેડલું મારા દલડા ને લૈ</br>
પાછી વળી ત્યારે બેડલુ નહી, બેડલુ નહી.

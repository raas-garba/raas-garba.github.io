# રમવા આવોને

અંબા તમને મારા સમ, મા તારી કળા નથી કંઈ કમ,</br>
...રમવા આવોને (2)

કીનખાબનો ઘાઘરોને ચોળી પહેરી તંગ</br>
રાતા પીળા ટપકાવાળી ઓઢણી કસુંબલ સંગ</br>
...રમવા આવોને

સેથો પૂર્યો સિંદુરનો ને ટીલડી રાતી રંગ</br>
હીરા જડેલ માની દામણીને નાકે જળુકે રંગ</br>
...રમવા આવોને

સાવરે સોનાની ચુડલી રે પહેરજો બાજુબંધ</br>
કેડે રૂડો કંદોરો પહેરજો પટ્ટો બાંધજો તંગ</br>
...રમવા આવોને

પાવાને ગઢથી કાળીને લાવજો બહુચરને લાવજો સંગ</br>
ખોડીયાર માને સાથે જો લાવશો રહેશે રૂડો રંગ</br>
...રમવા આવોને

લટકાળો કાનજી જોવાને આવશે લાડકડી રાધા સંગ</br>
ગોકુળ ગામની ગોપીઓ આવશે ગરબે જામશે રંગ</br>
...રમવા આવોને

પાયે તો માં ઝાંઝર વાગે નેપુર વાગે છમ</br>
તાળીયો એક તાલમાં વાગે ઘુઘરા વાગે ધમ</br>
...રમવા આવોને

આરાસુરથી અંબા પધાર્યા રાંદલ તુળજા સંગ</br>
સર્વે મળીને છંદ જ ગાયો બાળકનો રાખ્યો રંગ</br>
...રમવા આવોને

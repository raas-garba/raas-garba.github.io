# હું તો ગઇ'તી મેળે

હું તો ગઇ'તી મેળે</br>
મન મળી ગયું એની મેળામાં

હૈયું હણાઇ ને ગયું તણાઇ</br>
જોબન ના રેલામાં, મેળામાં... મેળામાં...

મેળે મેળાવનાર મેળો</br>
રંગ રેલાવનાર મેળો</br>
મૂલે મુલાવનાર મેળો</br>
ભૂલે ભુલાવનાર મેળો

ચિતડું ચકડોળ મારું આમ-તેમ ઘૂમતું</br>
ને આંખ લડી ગઇ અલબેલામાં

મેળામાં આંખના ઉલાળા</br>
મેળામાં પાયલ ઝણકાર</br>
કોઇના જાણે ત્યારે લાગે</br>
કાળજળે આંખ્યું ના માર

હેલાતા રંગે રેલામાં</br>
હૈયું હણાઇ ને ગયું તણાઇ</br>
જોબન ના રેલામાં, મેળામાં... મેળામાં...

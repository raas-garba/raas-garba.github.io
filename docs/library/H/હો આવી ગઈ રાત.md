# હો આવી ગઈ રાત

હો આવી ગઈ રાત તમે ભૂલો બધી વાત</br>
પ્રેમની આ મોસમ છે</br>
આવો મેરે પાસ</br>
રહે જાઓ મેરે સાથ</br>
પ્રેમની આ મોસમ છે

હો આવી ગઈ રાત

મિલ જાયે મુજકો અગર સાથ તેરા</br>
તો ભૂલો મેં સારા જહા

છોગાળા તારા, છબીલા તારા</br>
ઓ રે રંગીલા તારા રંગભેરુ જુએ તારી વાટ

આવી ગઈ રાત

છુપ છુપ કે તુજે દેખા મેં કરું સારી સારી રાત</br>
સારી સારી રાત</br>
મિલ જાયે મુજકો અગર સાથ તેરા</br>
તો ભૂલો મેં સારા જહાન

છોગાળા તારા..

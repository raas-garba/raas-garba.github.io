# હે માને આજે હું શ્રીફળ ચડાવું

હે માને આજે હું શ્રીફળ ચડાવું  
આવી આવી છે નોરતાની રાત  

જય જય આરાસુરની રાણી  
મા તું પાવાગઢ પ્રગટાણી  
મા તું સર્વશક્તિશાળી  
સૌનાં દુઃખડાં તું હરનારી  

દુઃખ હારીણી.... દુઃખ હારીણી  
હે માને આજે હું શ્રીફળ ચડાવું  

ઝગમગતો દિવડો શોભે તારા ગોખમાં  
ટમટમતા તારલા દીસે તારી આંખમાં  

દુઃખ હારીણી.... દુઃખ હારીણી  
હે માને આજે હું શ્રીફળ ચડાવું

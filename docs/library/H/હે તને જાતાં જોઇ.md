# હે તને જાતાં જોઇ

હે તને જાતાં જોઇ પનઘટની વાટે</br>
મારું મન મોહી ગયું

હે તારા રૂપાળા ગોરા ગોરા ઘાટે</br>
મારું મન મોહી ગયું હે તને...

કેડે કંદોરો કોટમાં દોરો</br>
તારા લહેરિયાની લાલ લાલ ભાતે</br>
મારું મન મોહી ગયું હે તને...

બેડલું માથે ને મહેંદી ભરી હાથે</br>
તારી ગાગરની છલકાતી છાંટે</br>
મારું મન મોહી ગયું હે તને...

રાસે રમતી, આંખને ગમતી</br>
પૂનમની રઢિયાળી રાતે</br>
મારું મન મોહી ગયું હે તને...

# હે મા અંબા જગદંબા

હે મા અંબા જગદંબા, દરશન દે મારી અંબા

માડી શક્તી દે ને, માડી ભક્તિ દે ને</br>
તારા ચરણોમાં રાખજે મા અંબા હે મા...

મારા આંગણીયે, મનનાં મંદિરીયે</br>
કુમકુમ તુ કરજે પગલાં ...ઓ

તારા વિના માડી સુના</br>
સુના મારા મંદિરને અંતરનાં ...ઓ

માડી ચરણે લે જે, માડી દુખડા હરજે</br>
તારા ચરણોમાં રાખજે મા અંબા હે મા...

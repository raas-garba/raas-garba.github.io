# હે તારી બાંકી રે

તારી બાંકી રે પાઘલડીનું ફૂમતું રે, મને ગમતું રે...</br>
આ તો કહું છું રે પાતળિયા તને અમથું.</br>
તારી બાંકી રે.....

તારા પગનું પગરખું ચમચમતું રે</br>
અને અંગનું અંગરખુ તસતસતું રે, મને ગમતું રે,</br>
આ તો કહું છું રે પાતળિયા તને અમથું.</br>
તારી બાંકી રે.....

પારકો જાણીને તને ઝાઝું શું બોલવુ?</br>
ને અણજાણ્યો જાણી તને મન શું ખોલવું?</br>
તને છેટો રે ભાળીને મને ગમતું રે</br>
આ તો કહું છું રે પાતળિયા તને અમથું.</br>
તારી બાંકી રે.....

હાથમાં ઝાલી ડાંગ કડિયાળી</br>
હરિયાળો ડુંગરો આવતો રે હાલી</br>
લીંબુની ફાડ જેવી આંખડીયું ભાળી</br>
શરમ મૂકીને તોયે થાઉં શરમાળી</br>
તારા રૂપનું તે ફૂલ મધમધતું રે, મને ગમતું રે,</br>
આ તો કહું છું રે પાતળિયા તને અમથું.</br>
તારી બાંકી રે.....

કોણ જાણે કેમ મારા મનની ભીતરમાં એવું તે ભરાયું શું</br>
એક મને ગમતો આભનો ચાંદલોને ને બીજો ગમતો તું!</br>
ઘરમાં, ખેતરમાં કે ધરતી ના થરમાં</br>
તારા સપનનમાં મન મારું રમતું રે, મને ગમતું રે,</br>
આ તો કહું છું રે પાતળિયા તને અમથું.</br>
તારી બાંકી રે.....

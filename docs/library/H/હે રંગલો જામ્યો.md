# હે રંગલો જામ્યો

હે રંગલો જામ્યો કાલિંદરીને ઘાટ</br>
છોગાળા તારા, હો રે છબીલા તારા</br>
હો રે રંગીલા તારા રંગભેરુ જુએ તારી વાટ</br>
... રંગલો જામ્યો કાલિંદરીને ઘાટ

હે હાલ્ય હાલ્ય હાલ્ય</br>
વહી જાય રાત વાતમાં ને, માથે પડશે પરભાત</br>
છોગાળા તારા, હો રે છબીલા તારા</br>
... રંગલો જામ્યો કાલિંદરીને ઘાટ

હે રંગરસીયા, હે રંગરસીયા</br>
તારો રાહડો માંડીને, ગામને છેવાડે બેઠાં</br>
કાના તારી ગોપલીએ,</br>
તારે હાટુ તો કામ બધા મેલ્યાં હેઠાં</br>
હે તને બરકે તારી જશોદ તારી માત</br>
છોગાળા તારા, હો રે છબીલા તારા</br>
હો રે રંગીલા તારા રંગભેરુ જુએ તારી વાટ</br>
... રંગલો જામ્યો કાલિંદરીને ઘાટ

મારા પાલવનો છેડલો મેલ,</br>
છોગાળા ઓ છેલ કે મન મારું મલકે છે</br>
એ હું મોરલો ને તું તો મારી ઢેલ</br>
હું છોડવો તું વેલ કે મન મારું ધડકે છે</br>
... રંગલો જામ્યો કાલિંદરીને ઘાટ

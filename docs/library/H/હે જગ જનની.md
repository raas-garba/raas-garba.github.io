# હે જગ જનની

હે જગ જનની, હે જગદમ્બા</br>
માત ભવાની શરણે લે જે

આદ્ય શક્તિ મા આદિ અનાદિ</br>
અરજી અંબા ઉરમાં ધર જે</br>
... હે જગ જનની, હે જગદમ્બા

કોઈના તીર નું નિશાન બની ને</br>
દિલ મારું તું વિંધાવા ના દે જે</br>
ઘા સહુ પણ ઘા કરું નહિ કોઈને</br>
એવી અંબા શક્તિ દે જે</br>
... હે જગ જનની, હે જગદમ્બા

હોય ભલે દુઃખ મેરુ સરખું</br>
રંજ એનો ન મને થાવા દે જે</br>
રજ સરખું દુઃખ જોઈ બીજા નું</br>
રોવા ને બે આંસુ દે જે</br>
... હે જગ જનની, હે જગદમ્બા.

આતમ કોઈનો આનંદ પામે તો</br>
ભલે સંતાપી લે મુજ આતમ ને</br>
આનંદ એનો અખંડિત રેહજો</br>
કંટક દે મને, પુષ્પો તેને</br>
... હે જગ જનની, હે જગદમ્બા

ધૂપ બનું હું સુગંધ તું લે જે</br>
રાખ બની ને ઉડી જવા દે જે</br>
બળું ભલે બાળુ નહિ કોઈ ને</br>
જીવન મારું સુગંધિત કર જે</br>
... હે જગ જનની, હે જગદમ્બા

અમૃત મળે કે ના મળે મુજને</br>
આશિષ અમૃતમય તું દે જે</br>
ઝેર જીવન ના હું પી જાણું</br>
પચાવવાની મને શક્તિ દે જે</br>
... હે જગ જનની, હે જગદમ્બા

દે જે શક્તિ મા ભક્તિ દે જે</br>
દુનિયા ના દુઃખ સહેવા કાજે</br>
દુર્લભ શક્તિ મા તારા ચરણમા</br>
અંબા મુજને તું ખોળે લે જે</br>
... હે જગ જનની, હે જગદમ્બા

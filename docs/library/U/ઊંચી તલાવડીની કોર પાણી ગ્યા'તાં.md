# ઊંચી તલાવડીની કોર પાણી ગ્યા'તાં

ઊંચી તલાવડીની કોર પાણી ગ્યા'તાં  
પાણી ભરતા રે જોયો સાહ્યબો  

બોલે અષાઢીનો મોર પાણી ગ્યાતાં  
પાણી ભરતા રે જોયો સાહ્યબો  

ગંગા જમની બેડલું ને કીનખાબી ઇંઢોણી  
નજરૂ ઢાળી હાલું તો'ય લાગી નજરૂ કોની...ઊંચી...  

વગડે ગાજે મુરલી ના શોર, પાણી ગ્યાતાં  
પાણી ભરતા રે જોયો સાહ્યબો

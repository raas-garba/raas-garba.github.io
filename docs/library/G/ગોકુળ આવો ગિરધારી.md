# ગોકુળ આવો ગિરધારી

અષાઢ ઉચ્ચારમ્, મેઘ મલ્હારમ્, બની બહારમ્, જલધારમ્</br>
દાદુર ડક્કારમ્, મયુર પુકારમ્, તડિતા તારમ્, વિસ્તારમ્</br>
ના લહી સંભારમ્, પ્યારો અપારમ્, નંદકુમારમ્ નિરખ્યારી</br>
કહે રાધે પ્યારી, હું બલિહારી, ગોકુળ આવો ગિરધારી

શ્રાવણ જલ બરસે, સુંદર સરસેં, બાદલ ભરસે, અંબરસેં</br>
તરુવર વિરિવરસે, લતા લહરસે, નદિયાં પરસે, સાગરસેં</br>
દંપતી દુઃખ દરસે, સેજ સમરસેં, લગત જહરસેં, દુઃખકારી</br>
કહે રાધે પ્યારી, હું બલિહારી, ગોકુળ આવો ગિરધારી

ભાદ્રવટ ભરિયા, ગિરિવર હરિયા, પ્રેમ પ્રસરિયા, તન તરિયા</br>
મથુરામેં ગરિયા, ફેરન ફરિયા, કુબજા વરિયા, વસ કરિયાં</br>
વ્રજરાજ વિસરિયા, કાજન સરિયા, મન નહિ ઠરિયા, હું હારી</br>
કહે રાધે પ્યારી, હું બલિહારી, ગોકુળ આવો ગિરધારી

આસો મહિનારી, આસ વધારી, દન દશરારી, દરશારી</br>
નવનિધિ નિહારી, ચઢી અટારી, વાટ સંભારી, મથુરારી</br>
ભ્રખુભાન દુલારી, કહત પુકારી, તમે થીયારી તકરારી</br>
કહે રાધે પ્યારી, હું બલિહારી, ગોકુળ આવો ગિરધારી

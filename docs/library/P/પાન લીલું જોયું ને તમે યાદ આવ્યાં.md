# પાન લીલું જોયું

પાન લીલું જોયું ... ને તમે યાદ આવ્યાં</br>
જાણે મૌસમનો પહેલો વરસાદ ઝીલ્યો રામ</br>
એક તરણું કોળ્યું ... ને તમે યાદ આવ્યાં

ક્યાંક પંખી ટહુક્યું ... ને તમે યાદ આવ્યાં</br>
જાણે શ્રાવણના આભમાં ઉઘાડ થયો રામ</br>
એક તારો ટમક્યો ... ને તમે યાદ આવ્યાં

જરા ગાગર છલકી ... ને તમે યાદ આવ્યાં</br>
જાણે કાંઠા તોડે છે કોઇ મહેરામણ હો રામ</br>
સહેજ ચાંદની છલકી ... ને તમે યાદ આવ્યાં

કોઇ ઠાલું મલક્યું ... ને તમે યાદ આવ્યાં</br>
જાણે કાનુડાના મુખમાં બ્રહ્માંડ દીઠું રામ</br>
કોઇ આંખે વળગ્યું ... ને તમે યાદ આવ્યાં

કોઇ આંગણે અટક્યું ... ને તમે યાદ આવ્યાં</br>
જાણે પગરવની દુનિયામાં શોર થયો રામ</br>
એક પગલું ઉપડ્યું ... ને તમે યાદ આવ્યાં

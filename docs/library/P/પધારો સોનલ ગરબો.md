# પધારો સોનલ ગરબો

હો સોનલ ગરબો લઇ માડી વિનવુ આજ રે</br>
સોળ સજી શણગાર માડી ગરબે ઘુમવા આજ</br>
પધારો.... સોનલ ગરબો

હો આરા તે સુરથી માડી રમવા પધારો</br>
નવલી તે નવરાત્રી માડી મા ઘુમવા પધારો મા</br>
ચોસઠ જોગણીઓને સંગે ગરબે રમવા આજ પધારો

હો.... પાવા તે ગઢથી માડી રમવા પધારો મા</br>
શારલોટ મંદિરમાં માડી ઘુમવા પધારો મા</br>
ભક્તજનોની સંગે માડી ગરબે ઘુમવા આજ પધારો

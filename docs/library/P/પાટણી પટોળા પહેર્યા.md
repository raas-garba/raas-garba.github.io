# પાટણી પટોળા પહેર્યા

પાટણી પટોળા પહેર્યા મારા વાલમા</br>
ચુડલા ચડાવ્યા રાતા ચોળ</br>
હે રુદિયામાં ટહુકે છે મોર

રૂમુ ઝુમુ રે સખી સૈયરોના સાથમાં</br>
ગરબે ઘૂમે રે લોલમ લોલ</br>
હે રુદિયામાં ટહુકે છે મોર

લીલુડા લહેરિયા લ્હેરે મારા વાલમા (2)</br>
ચીતરેલ કળાયેલ મોર</br>
ચામપાનંગે ચૂંદડી ચમકે મારા વાલમા</br>
સુરતની સોનેરી કોર,

હે રુદિયામાં ટહુકે છે મોર</br>
પાટણી પટોળા પહેર્યા

બિકાનેરી બંધાણી બંધાવી મારા વાલમા</br>
પહેરી છો નવી નક્કોર</br>
મઘમઘતો મોરલો મહેકે મારા વાલમા</br>
કોયલ કરે છે કલશોર,

હે રુદિયામાં ટહુકે છે મોર

# પોળ પછવાડે પરબડી

પોળ પછવાડે પરબડી ને વચમાં લીમડાનું ઝાડ</br>
હો રાજરાણી ઘૂમટો ઓઢીને તમે ક્યાં ગ્યા'તા

વગડા વચ્ચે વેલડી ને વચમાં સરોવર પાળ</br>
હો રાજરાણી ઘૂમટો ઓઢીને તમે ક્યાં ગ્યા'તા

ગામને પાદર ઢોલ ઢોલી</br>
ઢોલ વગાડે પગલો ઢોલી</br>
કાજળ આંજી આંખલડીને</br>
લહેરીયું છે લાલ હો રાજરાણી...

નાકે નથણી પગમાં ઝાંઝર</br>
હૈયે હેમનો હાર</br>
ચાલે ત્યારે ધરતી ધમકે</br>
આંખે રૂપનો ભાર</br>
પગ પર મોટી મોજલડી ને</br>
હંસી ચાલે ચાલ, હો રાજરાણી...

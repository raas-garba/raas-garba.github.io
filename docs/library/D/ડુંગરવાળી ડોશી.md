# ડુંગરવાળી ડોશી

ડુંગરવાળી ડોશી મા લાડ લડાવે</br>
લાડ લડાવે મૈયા લાડ વાકી લાવે

સુથારીનો બેટો મા બાજોઠ લઈ આવે</br>
પ્રેમ ધરીને માને બાજોઠે બેસાડે</br>
બાજોઠે બેસાડી માની આરતી ઉતારે</br>
આરતી ઉતારી માને શીશ નમાવે</br>
... ડુંગરવાળી ડોશી મા લાડ લડાવે

માળીડાનો બેટો મા ગજરા લઈ આવે</br>
પ્રેમ ધરીને માને ગજરા પહેરાવે</br>
ગજરા પહેરાવી માની આરતી ઉતારે</br>
આરતી ઉતારી માને શીશ નમાવે</br>
... ડુંગરવાળી ડોશી મા લાડ લડાવે

સોનીડાનો બેટો મા ઝાંઝરી લઈ આવે</br>
પ્રેમ ધરીને માને ઝાંઝરી પહેરાવે</br>
ઝાંઝરી પહેરાવી માની આરતી ઉતારે</br>
આરતી ઉતારી માને શીશ નમાવે</br>
... ડુંગરવાળી ડોશી મા લાડ લડાવે

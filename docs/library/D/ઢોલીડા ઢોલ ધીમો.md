# ઢોલીડા ઢોલ ધીમો

ઢોલીડા ઢોલ ધીમો, ધીમો વગાડ મા</br>
ધીમો વગાડ મા, નોરતાની રાતનો,</br>
જો જે રંગ જાયના............(2)

ધ્રુજે ના ધરતી તો રમઝટ કહેવાય ના,</br>
નોરતાની રાતનો જો જે રંગ જાયના .......ઢોલીડા

બજ્યો છે રાસને મચાવે શોર,</br>
થનગનતાં હૈયામાં નાચે છે મોર,</br>
પૂનમની રાતના દર્શન ભૂલાયના,</br>
ઢોલે રમતાં માડી રાત વીતી જાય ના .......ઢોલીડા

વાઘની સવારી ને હાથ હજારનાં,</br>
મોગરાની માળાને ફૂલડાંનો હાર,</br>
સોળે શણગાર ને દર્શન ભૂલાય ના .......ઢોલીડા

ગરબે રમે છે માડી નર ને નાર,</br>
રૂદિયો ખોલીને કરું છું પોકાર,</br>
ઘાયલ રૂદિયાને રોક્યું રોકાય ના ........ઢોલીડા

ગાવું ગવરાવવું એવી આશમાં,</br>
દાસ દોડીને આવે છે પાસમાં,</br>
દર્શન દઈને મા પાછા જવાય ના,</br>
ઢોલે રમતાં માડી રાત વીતી જાય ના ......ઢોલીડા

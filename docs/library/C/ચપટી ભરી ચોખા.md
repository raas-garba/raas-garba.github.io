# ચપટી ભરી ચોખા

ચપટી ભરી ચોખા ને ઘીનો છે દીવડો</br>
શ્રીફળની જોડ લઈએ રે....</br>
હાલો હાલો પાવાગઢ જઈએ રે... (2)

સામેની પોળથી સુથારી આવે,</br>
બાજોઠની જોડ લઈને રે હાલો....

સામેની પોળથી કસુંબી આવે,</br>
ચૂંદડીની જોડ અમે લઈએ રે હાલો...

સામેની પોળથી સોનીડો આવે,</br>
ઝાંઝરની જોડ અમે લઈએ રે હાલો....

સામેની પોળથી માળીડો આવે,</br>
ગજરાની જોડ અમે લઈએ રે હાલો....

સામેની પોળથી ઘાંચીડો આવે,</br>
દીવડાની જોડ અમે લઈએ રે હાલો....

ચપટી ભરી ચોખા ને ઘીનો છે દીવડો

# ચપટી રે ચોખા ને ચપટી રે કંકુ

ચપટી રે ચોખા ને ચપટી રે કંકુ....</br>
ચપટી રે ચોખા ને ચપટી રે કંકુ..

હે લઈને હાલો... હે લઈને હાલો બહુચર મા ને ધામ રે..</br>
બોલો બોલો બહુચર માં નું નામ રે

બોલો બહુચર માં, બોલો કાળકા રે માં, બોલો અંબાજી માં , બોલો આરાસુરી માં

ચપટી રે ચોખા ને ચપટી રે કંકુ...

હે આસોપાલવ ના તોરણ બંધાવો</br>
તોરણ બંધાવો.... તોરણ બંધાવો</br>
મારાગડે માડી તારા ફુલડા વેરાવુ..</br>
ફુલડા વેરાવ</br>
કે લઈ ને હાલો... કે લઈ ને હાલો બહુચર મા ને ધામ રે..</br>
બોલો બોલો બહુચર માં નું નામ રે...

બોલો બહુચર માં, બોલો દુર્ગા રે મા, બોલો અંબાજીમાં, બોલો આરાસુરી માં..

ચપટી રે ચોખા ને ચપટી રે કંકુ...

હે ઘીને પુરેલ ઓલા દિવડા પ્રગટાવો</br>
દિવડા પ્રગટાવો..... દિવડા પ્રગટાવો</br>
મારગડે માડી તારા ફુલડા વેરાવો...

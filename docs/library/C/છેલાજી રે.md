# છેલાજી રે

છેલાજી રે</br>
મારે હાટુ પાટણથી પટોળાં મોંઘાં લાવજો</br>
એમાં રૂડાં રે મોરલિયા ચિતરાવજો</br>
પાટણથી પટોળાં મોંઘાં લાવજો</br>
છેલાજી રે

રંગ રતુંબલ કોર કસુંબલ</br>
પાલવ પ્રાણ બિછાવજો રે</br>
પાટણથી પટોળાં મોંઘાં લાવજો</br>
છેલાજી રે

ઓલ્યા પાટણ શે'રની રે મારે થાવું પદમણી નાર</br>
ઓઢી અંગ પટોળું રે એની રેલાવું રંગધાર</br>
હીરે મઢેલા ચૂડલાની જોડ મોંઘી મઢાવજો રે</br>
પાટણથી પટોળાં મોંઘાં લાવજો</br>
છેલાજી રે

ઓલી રંગ નીતરતી રે મને પામરી ગમતી રે</br>
એને પહેરતાં પગમાં રે પાયલ છમછમતી રે</br>
નથણી લવિંગિયાં ને ઝૂમખાંમાં મોંઘાં મોતી મઢાવજો રે</br>
પાટણથી પટોળાં મોંઘાં લાવજો</br>
છેલાજી રે

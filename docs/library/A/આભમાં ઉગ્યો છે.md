# આભમાં ઉગ્યો છે

આભમાં ઉગ્યો છે ચાંદલો ને</br>
સાથે રાસડાની રમઝટ હો</br>
ઢોલકના તાલ પર પડે છે તાળી ને</br>
ગરબાની રમઝટ હો

નવનવ દિવડાની જ્યોત પ્રગટાવી (૨)</br>
નવનવ રંગોની ગરબી સજાવી (૨)</br>
સોના રૂપાની માડી થાળી સજાવી ને</br>
દિવડાંની ઝગમગ હો.... આભમાં

નવરંગ ફૂલોની માળા બનાવી (૨)</br>
નવનવ રંગોની આરતી સજાવી (૨)</br>
ભાવભર્યા હૈયે મે આરતી ઉતારી કે</br>
આરતી ઝગમગ હો.... આભમાં

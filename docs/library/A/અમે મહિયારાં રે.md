# અમે મહિયારાં રે

અમે મહિયારાં રે, ગોકુળ ગામનાં</br>
મારે મહી વેચવાને જાવાં ...મહિયારાં રે

મથુરાની વાટ મહીં વેચવાને નીસરી</br>
નટખટ એ નંદકિશોર માગે છે દાણ જી</br>
હે....મારે દાણ દેવા, નહીં લેવા ...મહિયારાં રે

માવડી જશોદાજી કાનજીને વારો</br>
દુ:ખડા હજાર દિયે નંદજીનો લાલો</br>
હે....મારે દુ:ખ સહેવા, નહીં કહેવા ...મહિયારાં રે

યમુનાને તીર વા'લો વાંસળી વગાડતો</br>
ભુલાવી ભાન સાન ઉંઘતી જગાડતો</br>
હે....મારે જાગી જોવું ને જાવું ...મહિયારાં રે

નરસિંહનો નંદકિશોર લાડકડો કાનજી</br>
ઉતારે આતમથી ભવ-ભવનો ભાર જી</br>
નિર્મળ હૈયાંની વાત કહેતા ...મહિયારાં રે

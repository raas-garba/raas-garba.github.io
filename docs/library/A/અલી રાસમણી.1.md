# અલી રાસમણી

અલી રાસમણી રમવા આવ</br>
કે આવ્યા માનાં નોરતા

અલી ગરબામાં દિવડો પ્રગટાવ</br>
ગોરી અબાંને શીષ નમાવ</br>
કે આવ્યા માનાં નોરતા

અલી રાસમણી...

આવી આસોની અજવાળી રાત</br>
એમાં મઘમઘતા તારલાની ભાત</br>
તારી પાનીએ મેંદી મુકાવ</br>
તારા રૂદિયાને રમણે ચડાવ</br>
હે આવ્યા માનાં નોરતા

અલી રાસમણી...

તારી કાયાને શણગારી મેલ</br>
સખી ચાચરના ચોકમાં ખેલ

તારા ગરબાને ફુલડે વધાવ</br>
એમા ચૌદ બ્રહમાંડ રચાવ</br>
હે આવ્યા માનાં નોરતા

અલી રાસમણી...

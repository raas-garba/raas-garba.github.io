# અલી રાસમણી

અલી રાસમણી રાસમણી રાસમણી રે</br>
આવ્યા છે માના નોરતા રે

ગોરી ગરબામા દિવડો મૂક રે</br>
ગોરી અબાંને ચરણે ઝુક રે.</br>
માના ગરબાને ફુલડે વધાવ ...અલી રાસમણી

આવી આસોની અજવાળી રાત રે</br>
એમાં દુધમલીયા તારાની ભાત રે</br>
તારી પેનીએ મેંદી મૂકાવ ...અલી રાસમણી

તારી કાયાની શણગારી વેલ રે</br>
સખી ચાચરના ચોકમાં ખેલ રે</br>
તારા રુદિયાને રમણે ચડાવ ...અલી રાસમણી

મન મંદિરમાં અંબા થઇ નાચ રે</br>
માઇ મંડળમાં બહુચર થઇ નાચ રે</br>
તારા ચૌદે બ્રહમાંડ મચાવ રે ...અલી રાસમણી

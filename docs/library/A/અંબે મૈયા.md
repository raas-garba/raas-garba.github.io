# અંબે મૈયા

અંબે મૈયા મને તારું શરણું મળે...</br>
મારું જીવન માતાની ભક્તિમાં રહે...</br>
મૈયા અમર રહે...

મૈયા ઓ અંબે મૈયા...

હો... મારું જીવન છે તારું ઓ મૈયા</br>
તારા હાથમાં છે મારી નૈયા

આવી હું તારે દ્વાર સ્વર મંદિર સથવાર</br>
આ ભાવ તારી સેવાનો લ્હાવો રહે</br>
લહાવો રહે...

મૈયા... ઓ અંબે મૈયા

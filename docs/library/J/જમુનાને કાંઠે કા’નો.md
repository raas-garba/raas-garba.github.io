# જમુનાને કાંઠે કા'નો

જમુનાને કાંઠે કા'નો વાંસળી વગાડતો</br>
સૂર એના એવા રેલાય રે...</br>
વાંસળીના સૂર સૂણી, ગોપીઓ ભાન ભૂલી</br>
સૂર એના એવા રેલાય રે...

ઝરમર ઝરમર મેહુલિયો વરસે</br>
રાધા સંગ કાનો ભીંજાય રે</br>
કાળી કાળી વાદળીમાં વીજળી ઝબૂકી</br>
બારે મેઘ આજે મંડાય રે

મનમાં મારા ઉમંગ એવો જાગ્યો</br>
મીરા થઇ તુજમાં સમાઇ જાઉં</br>
વિરહની વેદના હવે સહાય ના</br>
ઓ શ્યામ મુજને સમાવી લે

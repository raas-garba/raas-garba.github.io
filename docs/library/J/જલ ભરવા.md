# જલ ભરવા

જલ ભરવાને કેમ જાઉ જશોદા મૈયા (૨)</br>
આ વાટે ને ઘાટે વા'લો પાણીડા માગે</br>
હે... લાજે દેખે ને કેમ જાવું જશોદા મૈયા

લાજ તજીને વાલો ને લપ બન્યો છે</br>
હે... મારે નિર્ણય કેમ થાવું જશોદા મૈયા

સોના સરીખા બેડલા અમારા...</br>
હે... ઇંઢોણી રતન જણાવું જશોદા મૈયા

બાઈ મીરાં કહે પ્રભુ ગિરિધર નાગર</br>
હે... ચરણકમળ ચિત્ત લાલ જશોદા મૈયા

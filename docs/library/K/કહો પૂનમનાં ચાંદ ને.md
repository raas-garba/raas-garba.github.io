# કહો પૂનમનાં ચાંદ ને

કહો પૂનમનાં ચાંદને આજે ઉગે આથમની ઓર

મારા મનડાના મીત,</br>
મારા જીવન સંગીત લઈને આવ્યા છે મારી પ્રીત</br>
કહો પૂનમનાં ચાંદને...

આજે અમારાજીવનનો આ કેટલો સુંદર દિવસ છે.</br>
આજે અમે રમશું પ્રીતમની સાથે હાથોમાં રાખીને હાથ</br>
કહો પૂનમનાં ચાંદને...

પૂનમની રાત થંભી જાશે તો ઝૂલશું ફૂલડાંની સેજ કરીને</br>
નદીઓનાં નીરમાં નાવલડી તારશું ગીતોનાં ગુંજન કરીને</br>
કહો પૂનમનાં ચાંદને...

ક્યારે ફરી આવશે અવસર અનેરો પ્રીતમની સંગે રહેવાનો</br>
ક્યારે ફરી મળશું રઢિયાળી રાતમાં, મનડામાં પ્રીત ભરીને</br>
કહો પૂનમનાં ચાંદને...

# કૃષ્ણ ભગવાન ચાલ્યા

કૃષ્ણ ભગવાન ચાલ્યા દ્વારિકાને કઈ</br>
લીધો રે મણિયારા વાળો વેશ</br>
કે હોવે હોવે, લીધો મણિયારા વાળો વેશ

કે હું તો તને વારી જાઉં જી હો મણીયારા</br>
કે હૂતો તને.. વારુ લ્યા જી હો મણિયારા..

હે હે રાધારાણી રે બેઠા મેડીએ ને કાંઈ</br>
જુવે મણિયારા તારી વાટ હો મણિયારો...</br>
હે મણિયારો મણિયારો હું તો</br>
શું રે કરું રે મારો મણિયારો સે નાનેરો બાળ</br>
કે હોવે હોવે, મણિયારો નાનેરો બાળ

કે હું તો તને... વારુ લ્યા જી હો મણિયારા (૨)</br>
દ્વારિકાની શેરીઓમાં ઘુમે રે મણિયારો (૪)</br>
હે ચૂડલા વેચે ઓલો નંદનો દુલારો (૨)</br>
હો ચૂડલા વેચે ઓલો નંદનો દુલારો (૨)</br>
હો દ્વારિકાની શેરીઓમાં ઘૂમે રે મણિયારો...

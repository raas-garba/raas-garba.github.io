# કુકડા તારી બોલી

બહુચરમાનાં ડેરા પાછળ કૂકડે કૂક બોલે રે,</br>
ઓ કુકડા તારી બોલી મને મીઠી મીઠી લાગે

ભકતોની તું રક્ષા કરતી, માતા બિરદાળી,</br>
માડી માતા બિરદાળી,</br>
તારા ચરણે આવે તેના પાપો નાખે બાળી</br>
ઓ કુકડા તારી બોલી મને મીઠી મીઠી લાગે

હે કાનમાં કુંડળ શોભે, તારા પગમાં ઝાંઝર બોલે</br>
તારા પગમાં ઝાંઝર બોલે</br>
માડી તારા ચણીયા ચોળી ઘુમે ઝાકમ જોળે</br>
ઓ કુકડા તારી બોલી મને મીઠી મીઠી લાગે

ત્રણે લોકની રાણી, તું તો સર્વ શક્તિશાળી</br>
તું તો સર્વ શક્તિશાળી</br>
નમી નમી પાયે લાગું ભકતોની તું પ્યારી</br>
ઓ કુકડા તારી બોલી મને મીઠી મીઠી લાગે

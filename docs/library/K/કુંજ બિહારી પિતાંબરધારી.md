# કુંજ બિહારી પિતાંબરધારી

કુંજ બિહારી પિતાંબરધારી મારો મારગડો રોકે ગિરધારી રે</br>
વ્હાલમીયાને કહેજો તમે જીત્યા ને હું હારી રે

કાલીન્દીને તીરે ગોપ ગોવાલણી</br>
રાસ રમાડે, રાસ રમાડે, ગિરધારી રે

મારગ રોકીને કાનજીએ મારી</br>
નવરંગ ચુંદડી, નવરંગ ચુંદડી, ઝાલી રે

હે.... હું તો લાજ શરમથી શરમાણી રે</br>
વ્હાલમીયાને કહેજો તમે જીત્યા ને હું હારી રે

કુંજ બિહારી પિતાંબરધારી મારો મારગડો રોકે ગિરધારી રે

મધરાતે મણીધરની મોરલીનાં નાદે હું તો</br>
ભરી નીંદરમાંથી, ભરી નીંદરમાંથી, જાગી રે

મોરલીનાં નાદે મારૂ મનડું મોહ્યું</br>
મારી નીંદર, મારી નીંદર, વેરણ થઈ રે

હે.... હું તો જાણી જાણી તોય અજાણી રે</br>
વ્હાલમીયાને કહેજો તમે જીત્યા ને હું હારી રે

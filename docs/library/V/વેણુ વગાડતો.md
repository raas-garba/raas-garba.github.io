# વેણુ વગાડતો

વેણુ વગાડતો ... વેણુ વગાડતો</br>
વેણુ વગાડતો, ગાયો હંકારતો</br>
આયો જશોદાનો કાનડો</br>
... વેણુ વગાડતો

માથે છે મોરપિચ્છ, કેડે કંદોરો</br>
હળવેથી હળવેથી કાનુડો આવતો</br>
પનઘટની કેડીએ મારગડો રોકતો</br>
... વેણુ વગાડતો

સહિયર સૌ કાનને હેતે રમાડ્યા</br>
મટકીથી મટકીથી મહીડા ચુરાવ્યા</br>
મહીડા ચુરાવીને દલડા ચુરાવતો</br>
... વેણુ વગાડતો

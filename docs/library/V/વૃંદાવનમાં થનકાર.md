# વૃંદાવનમાં થનકાર

વૃંદાવનમાં થનકાર થનક થૈ થૈ થૈ</br>
રાધાકૃષ્ણ રમે રાસ ગોપી લૈ લૈ લૈ</br>
બીજું કૈં નહીં કૈં નહીં વૃંદાવનમાં...

નૂપુરચરણ કનકવરણ ઝાંઝર જોડો</br>
ઘુંઘરીયાળો કટી ઓપે કંદોરો</br>
મોરમુકુટ મણી વાંકડો અંબોડો</br>
કુંડલકાન, ભ્રુકુટી તાન, નૈનબાણ કંપમાન</br>
તાળી લૈ લૈ લૈ વૃંદાવનમાં...

વાગે તાલ ને કરતાલ સંગ તાળી</br>
કોઇ તંબુરો ને કોઇ મૃદંગવાળી</br>
મદનગાન મુખ્ય ગાયે વનમાળી</br>
બોલે બૈન, સુધા સૈન, મોહન નૈન, પ્રગટ ચૈન</br>
હ્રદય દૈ દૈ દૈ વૃંદાવનમાં...

મુકુટમાંહી રૂપ દીઠું રાધાએ</br>
મનમાં માનિની વિમાસણ થાયે</br>
હુંથી છાની બીજા છે મુકુટ માંહે</br>
બહુ વ્હાલી, હઠ ઝાલી, ઊઠી ચાલી</br>
દયા પ્રભુ જય જય જય વૃંદાવનમાં...

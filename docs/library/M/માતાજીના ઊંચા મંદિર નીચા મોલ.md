# માતાજીના ઊંચા મંદિર નીચા મોલ

માતાજીના ઊંચા મંદિર નીચા મોલ જરૂખડે દીવા બળે રે લોલ</br>
રાધાગોરી ગરબે રમવા આવો સાહેલી સહુ ટોળે વળી રે લોલ</br>
માતાજીના

આવી રૂડી નવરાત્રીની રાત કે બાળ સહુ રાસે રમે રે લોલ</br>
ચામુંડા માડી ગરબે રમવા આવો કે બાળ તારા વીનવે રે લોલ</br>
માતાજીના

માતાજીને શોભે છે શણગાર સોળ કે પગલે કંકુ ઝરે રે લોલ</br>
રાંદલમા ગરબે રમવા આવો કે મુખડે ફૂલ ઝરે રે લોલ</br>
માતાજીના

માડી તારું રૂપ અનુપમ સોહે કે જોઈ મારી આંખો ઠરે રે લોલ</br>
બહુચર મારી ગરબે રમવા આવો કે મુખડે અમી ઝરે રે લોલ</br>
માતાજીના

# મા પાવા તે ગઢથી ઉતર્યા

મા પાવા તે ગઢથી ઉતર્યા મા કાલી રે</br>
વસાવ્યું ચાંપાનેર પાવાગઢવાળી રે</br>
મા ચાપા તેને ના ચાર ચૌટા, મહાકાલી રે</br>
સોનીડે માંડ્યા હાથ, પાવાગઢ વાળી રે,</br>
મા સોનીડો લાવે રૂડાં જુમખાં, મહાકાળી રે</br>
મારી અંબે માને કાજ, પાવાગઢ વાળી રે

મા પાવા તે ગઢથી

મા માળીડો આવે મલપતો, મા કાલી રે</br>
એ લાવે ગજરાની જોડ, પાવાગઢ વાળી રે

મા પાવા તે ગઢથી

મા કુંભારી આવે મલપતો, મા કાલી રે</br>
એ લાવે ગરબાની જોડ, પાવાગઢ વાળી રે

મા પાવા તે ગઢથી

મા સુથારી આવે મલપતો, મા કાલી રે</br>
એ લાવે બાજોઠની જોડ, પાવાગઢ વાળી રે

મા પાવા તે ગઢથી

# મોગલ આવે

મોગલ આવે... નવરાત રમવા, કેવા કેવા વેશે મા</br>
કેવા કેવા વેશે...</br>
તું સહુને દેખે માતાજી...</br>
તું સહુને દેખે માતાજી, તુજને કોઈ ન ભાળે...</br>
મોગલ આવે... નવરાત રમવા...

રે મોગલ આવે નવરાત રમવા કેવા કેવા વેશે

તારા ઝણણ ઝણણ ઝાંઝર ઝમકે</br>
ખણણ ખણણ કાંબી મા... ખણણ ખણણ કાંબી</br>
તારા હાથે હેમનાં કડાં ઝળુંબે, લટું મોકળી લાંબી</br>
ભાતળીયું ભેળી માતાજી</br>
ભાતળીયું ભેળી મા વાળી લેશે ને કાંઇ દેશે</br>
મોગલ આવે... નવરાત રમવા...

તારા ડગલે ને પગલે કંકુ ઝરતા,</br>
આખો મારગ રાતો મા</br>
ચાંદનીનો ચંદરવો માથે, લીલી કોરની રાતો મા</br>
"દાન" કહે મરછરાળી મોગલ, સદાય સહાય રહેશે</br>
મોગલ આવે... નવરાત રમવા...

# મેલી દીયોને ગીરધારી

મેલી દીયોને ગીરધારી રે મારગડો મારો..</br>
મેલી દીયોને ગીરધારી..</br>
તારે મારગડે આવતાં ને જાતાં,</br>
ઓલ્યો દાણ માગે કાનુંડો દાણી.</br>
મારગડો મારો..

ખારા સમુદરિયામાં મીઠી એક વીરડી,</br>
એનાં પાણી ભરે છે પનિહારી.</br>
મારગડો મારો..

સાસુ ને સસરા મારા ઘરમાં છે બૂઢા,</br>
ઓલી નણદલ છે રે નઠારી.</br>
મારગડો મારો..

બાઈ મીરાં કહે પ્રભુ ગીરધર નાગર,</br>
તમે જીત્યા ને હું હારી..</br>
મારગડો મારો..

# માડી તારા વિના

માડી તારા વિના... ગરબો સૂનો રે  
માડી તારા વિના... ગરબો સૂનો રે  
તમે આવો માડી, પધારો માડી  

આવો ને માડી... પધારો માડી...  

માડી તારા વિના... ગરબો સૂનો રે  
તમે આવો માડી, પધારો માડી  

હો અંબે માવડી રે...  

સૈયર બોલાવે રમવા આવો  
ગરબો ચડ્યો છે આજે રમણે રે  

હો અંબે માવડી રે...  

આવી રૂડી નોરતાની રાત  
સૈયર સંગાથે આવો માવલડી  
માડી તારા વિના... ગરબો સૂનો રે

# માડી તારા મંદિરીયામાં

માડી તારા મંદિરીયામાં ઘંટરવ વાગે</br>
હે ઉંચા ઉંચા ડુંગરીયામાં માડી તું બિરાજે</br>
જગદંબે... બોલો અંબે...

લાલ ચટક ચુંદડી સોહે સાવજે અસવારો</br>
તેજ ભર્યુ મુખડુ જોઇ જાવુ વારે વારે</br>
હે... ચાચર ચોક ઉતર્યા માડી ગરબો ગાવા આજે</br>
જગદંબે... બોલો અંબે...

માડી તારી રૂપ ઘણા નામ તો હજાર છે</br>
જગમાં તારો મહિમા મોટો તું તો તારણહાર છે</br>
ભીડ ભાંગી ભક્તો કેરી આશીષ દેતી જાજે</br>
જગદંબે... બોલો અંબે...

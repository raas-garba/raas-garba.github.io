# માએ ગરબો કોરાવ્યો

આરાસુરની અંબિકા, તન ઝૂલે હીંડોળા ખાટ,</br>
અવનીના દરબારમાં રમવા નીસર્યા માત.

મા એ ગરબો કોરાવ્યો ગગન ગોખમાં રે,</br>
સજી સોળ રે શણગાર, મેલી દિવડાં કેરી હાર.</br>
...હે માએ ગરબો કોરાવ્યો ગગન ગોખમાં રે

ગબ્બરની લઇ માંડવી માથે ઘૂમતી મોરી માત</br>
ચુંદલડીમાં ચમકે ઝાઝી રૂપલે મઢી રાત.</br>
જોગમાયાને અંગ નર્યો નીતરે ઉમંગ</br>
રમે જોગણીયો સંગ રૂડો અવસરનો રંગ

માએ પાથર્યો પરકાશ ચૌદ લોકમા રે</br>
...હે માએ ગરબો કોરાવ્યો ગગન ગોખમાં રે

ચારે જગનો ચડલો માનો સોળ કળાનો વાન</br>
અંબાના અણસારા વીના હાલે નહિ પાન</br>
માના રૂપની નહિ જોડ, એને રમવાનાં બહુ કોડ</br>
માંડે ગરબા કેરી હોડ, રૂડો અવસરનો રંગ

માએ ગરબો ચગાવ્યો ચાચર ચોકમા રે .</br>
...હે માએ ગરબો કોરાવ્યો ગગન ગોખમાં રે

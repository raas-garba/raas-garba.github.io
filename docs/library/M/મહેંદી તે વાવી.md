# મહેંદી તે વાવી

મહેંદી તે વાવી માળવે ને</br>
એનો રંગ ગયો ગુજરાત રે</br>
મહેંદી રંગ લાગ્યો રે

નાનો દિયરીયો લાડકો ને,</br>
કાંઇ લાવ્યો મહેંદીનો છોડ રે ...</br>
મહેંદી રંગ લાગ્યો રે...

વાટી ઘૂંટીને ભર્યો વાટકો ને</br>
ભાભી રંગો તમારા હાથ રે ...</br>
મહેંદી રંગ લાગ્યો રે ...

હે... લાંબો ડગલો, મૂછો વાંકડી, શિરે પાઘડી રાતી</br>
બોલ બોલતો તોળી-તોળી છેલછબીલો ગુજરાતી</br>
હે.. તન છોટુ પણ મન મોટું, છે ખમીરવંતી જાતી</br>
ભલે લાગતો ભોળો, હું છેલછબીલો ગુજરાતી

હાથ રંગીને વીરા શું રે કરું?</br>
એનો જોનારો પરદેશ રે ...

મહેંદી રંગ લાગ્યો રે ...</br>

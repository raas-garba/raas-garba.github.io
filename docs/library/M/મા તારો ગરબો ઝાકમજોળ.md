# મા તારો ગરબો ઝાકમજોળ

હે મા તારો ગરબો ઝાકમજોળ, ઘૂમે ગોળ ગોળ,</br>
પાવાગઢની પોળમાં રે લોલ..</br>
મા તારી ચૂંદડી રાતીચોળ, ઊડે રંગચોળ,</br>
પાવાગઢની પોળમાં રે લોલ..

હે માડી ગરબે ઘૂમે, હે સજી સોળ શણગાર,</br>
માડી તારા ચરણોમાં પાવન પગથાર.</br>
મા તારે ગરબે ફૂલનો હિંડોળ, મોંઘો અણમોલ,</br>
પાવાગઢની પોળમાં રે લોલ..

હે મા તારો ગરબો ઝાકમજોળ, ઘૂમે ગોળ ગોળ,</br>

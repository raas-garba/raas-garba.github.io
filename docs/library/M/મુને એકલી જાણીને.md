# મુને એકલી જાણીને

મુને એકલી જાણીને ક્હાને છેડી રે...</br>
પછી કહી દઉં જશોદાના કાનમાં</br>
મારો મારગડો છોડીને હાલતો થા...</br>
પછી કહી દઉં જશોદાના કાનમાં

મેળામાં મળવા હાલી, મારી સરખી સૈયરને</br>
મેળામાં મળી ગયો પેલો રે તોફાની ક્હાન,</br>
મારો છેડલો ન ઝાલ, તને કહી દઉં છું...</br>
પછી કહી દઉં જશોદાના કાનમાં

મુને એકલી જાણીને ક્હાને છેડી રે...

બેડલું લઇને હું તો સરોવર ગઇ'તી</br>
પાછું વળીને જોયું, બેડલું ચોરાઇ ગયું,</br>
મારા બેડલાનો ચોર, મારે કેમ લેવો ખોળી...</br>
દઇ દે બેડલું રે ઓ મારા ક્હાનજી.

# એક વાર બોલું

એક વાર બોલું કે,  
બે વાર બોલું કે,  
ત્રણ વાર બોલું ઓ મા,  
મા તમે ગરબે રમવા આવજો...  
ગરબે રમવા આવજો માડી,  
દર્શન દેવા આવજો  

ઉતારા દેશું રે મા તને મેડીના મોલના,  
એક વાર આવીને મારે મંદિરીયે,  
ઉતારા કરતા જાવ  
મા તમે ગરબે રમવા આવજો...  
ગરબે રમવા આવજો માડી,  
દર્શન દેવા આવજો  

ભોજન દેશું રે મા તને મોંઘાને ભાવતા,  
એક વાર આવીને મારે મંદિરીયે,  
ભોજન કરતાં જાવ  
મા તમે ગરબે રમવા આવજો...  
ગરબે રમવા આવજો માડી,  
દર્શન દેવા આવજો

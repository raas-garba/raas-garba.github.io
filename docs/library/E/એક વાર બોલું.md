# એક વાર બોલું

એક વાર બોલું કે,</br>
બે વાર બોલું કે,</br>
ત્રણ વાર બોલું ઓ મા,</br>
મા તમે ગરબે રમવા આવજો...</br>
ગરબે રમવા આવજો માડી,</br>
દર્શન દેવા આવજો

ઉતારા દેશું રે મા તને મેડીના મોલના,</br>
એક વાર આવીને મારે મંદિરીયે,</br>
ઉતારા કરતા જાવ</br>
મા તમે ગરબે રમવા આવજો...</br>
ગરબે રમવા આવજો માડી,</br>
દર્શન દેવા આવજો

ભોજન દેશું રે મા તને મોંઘાને ભાવતા,</br>
એક વાર આવીને મારે મંદિરીયે,</br>
ભોજન કરતાં જાવ</br>
મા તમે ગરબે રમવા આવજો...</br>
ગરબે રમવા આવજો માડી,</br>
દર્શન દેવા આવજો

# એક વણઝારી ઝૂલણાં

એક વણઝારી ઝૂલણાં ઝૂલતી'તી,</br>
મારી અંબેમાંના ઝૂલણાં ઝૂલતી'તી.

માએ પહેલે પગથીયે પગ મૂક્યો,</br>
માની પાની સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.

માએ બીજે પગથીયે પગ મૂક્યો,</br>
માનાં ઘૂંટણ સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ ત્રીજે પગથીયે પગ મૂક્યો</br>
માનાં ઢીંચણ સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ ચોથે પગથીયે પગ મૂક્યો,</br>
માના સાથળ સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ પાંચમે પગથીયે પગ મૂક્યો,</br>
માની કેડ સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ છઠ્ઠે પગથીયે પગ મૂક્યો,</br>
માની છાતી સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ સાતમે પગથીયે પગ મૂક્યો,</br>
માનાં ગળાં સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ આઠમે પગથીયે પગ મૂક્યો,</br>
માનાં કપાળ સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

માએ નવમે પગથીયે પગ મૂક્યો,</br>
માનાં માથાં સમાણાં નીર મોરી માત;</br>
વણઝારી ઝૂલણાં ઝૂલતી'તી.</br>
એક વણઝારી...

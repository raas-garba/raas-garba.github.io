# એક લાલ દરવાજે

એક લાલ દરવાજે તંબુ તાણીયા રે લોલ

અમદાવાદી નગરી</br>
એની ફરતે કોટે કાંગરી</br>
માણેકલાલની મઢી</br>
ગુલઝારી જોવા હાલી

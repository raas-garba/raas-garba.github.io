# એક લાલ દરવાજે

એક લાલ દરવાજે તંબુ તાણીયા રે લોલ  

અમદાવાદી નગરી  
એની ફરતે કોટે કાંગરી  
માણેકલાલની મઢી  
ગુલઝારી જોવા હાલી  

હે વઉ તમે નઉ જશો જોવાને  
ત્યાં બાદશો બડો મિજાજી  

એક લાલ દરવાજે તંબુ તાણીયા રે લોલ  

સીદી સૈયદની જાળી  
ગુલઝારી જોવા હાલી  
કાંકરિયાનું પાણી  
ગુલઝારી જોવા હાલી  

હે વઉ તમે નઉ જશો જોવાને  
ત્યાં બાદશો બડો મિજાજી  

એક લાલ દરવાજે તંબુ તાણીયા રે લોલ  

ત્રણ દરવાજા માંહી  
માં બિરાજે ભદ્રકાળી  
માડીના મંદિરીયે  
ગુલઝારી જોવા હાલી  

હે વઉ તમે નઉ જશો જોવાને  
ત્યાં બાદશો બડો મિજાજી  

એક લાલ દરવાજે તંબુ તાણીયા રે લોલ

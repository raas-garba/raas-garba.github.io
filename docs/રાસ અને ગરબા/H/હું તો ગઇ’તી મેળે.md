# હું તો ગઇ'તી મેળે

હું તો ગઇ'તી મેળે  
મન મળી ગયું એની મેળામાં  

હૈયું હણાઇ ને ગયું તણાઇ  
જોબન ના રેલામાં, મેળામાં... મેળામાં...  

મેળે મેળાવનાર મેળો  
રંગ રેલાવનાર મેળો  
મૂલે મુલાવનાર મેળો  
ભૂલે ભુલાવનાર મેળો  

ચિતડું ચકડોળ મારું આમ-તેમ ઘૂમતું  
ને આંખ લડી ગઇ અલબેલામાં  

મેળામાં આંખના ઉલાળા  
મેળામાં પાયલ ઝણકાર  
કોઇના જાણે ત્યારે લાગે  
કાળજળે આંખ્યું ના માર  

હેલાતા રંગે રેલામાં  
હૈયું હણાઇ ને ગયું તણાઇ  
જોબન ના રેલામાં, મેળામાં... મેળામાં...

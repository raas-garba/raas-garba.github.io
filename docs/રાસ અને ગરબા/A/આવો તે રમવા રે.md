# આવો તે રમવા રે

    નેહ નીસરતી આંખલડી ને અધરે અમૃતધાર
    ગરબો લઈ સુર માવડી મારે
    કુમકુમ પગલે પધાર... હા...

    આવો તે રમવા રે, મા ગરબે ઘૂમવા ને

    ગરબે ઘૂમતા રે...(૨)

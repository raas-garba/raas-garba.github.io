# અંબાજી હિંચકે ઝુલે રે

અંબાજી હિંચકે ઝુલે રે, મા ઝુલે રે આરાસુર ધામમાં  
નવરંગા ઝુલણે ઝુલે રે, મા ઝુલે રે આરાસુર ધામમાં  
અંબાજી હિંચકે ઝુલે રે...
